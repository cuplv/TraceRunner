TraceMsg

Related Docs: object TraceMsg | package TraceMsgContainer

final case class TraceMsg(type: MsgType, messageId: Long, threadId: Long, app: Option[AppMsg] = None, isActivityThread: Boolean, oneofMessage: OneofMessage = ...) extends GeneratedMessage with Message[TraceMsg] with Updatable[TraceMsg] with Product with Serializable

Defines a single message sent by the instrumented App The message can be of several types: AppMsg, CallinMsg, CallbackMsg.

type

Identifies the type of the message

messageId

Defines the position of the message in the trace

threadId

id of the thread on which the message was recorded
